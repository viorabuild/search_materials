# Contributing

1. **Environment**
   - Используйте Python 3.10+ и виртуальное окружение (`python -m venv .venv`).
   - Установите зависимости через `pip install -r requirements.txt`.

2. **Стиль кода**
   - Соблюдайте типы и аннотации, избегайте лишнего глобального состояния.
   - Предпочитайте модульные функции и классы с понятными докстрингами.
   - Для форматирования можно использовать `ruff`/`black` (не включены в зависимости по умолчанию).

3. **Secrets**
   - Не коммитите `.env`, `credentials/*.json`, файлы кэша.
   - Используйте шаблон `.env.example` и README в `credentials/`.

4. **Тестирование**
   - Локально проверяйте выполнение основных сценариев: `ConstructionAIAgent.process_command`, веб-интерфейс `./run_unified.sh`.
   - При изменении работы с Google Sheets используйте тестовую копию таблицы.

5. **Вклад**
   - Перед Pull Request обновите документацию, если меняется поведение API/конфигурации.
   - Обновите `docs/changelog.md`.
